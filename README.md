# Math Notes

Temporary notes in mystmd that will eventually go into
[TPR](https://theoretical-physics.com).

Web version: https://math-notes-e2r.pages.dev/

## Build and serve html

```
pixi r start
```

## Build pdf

Using TeX:
```
pixi r tex
open _build/exports/paper_tex/paper.pdf
```

Using Typst:
```
pixi r typst
open _build/exports/paper_typst/paper.pdf
```
